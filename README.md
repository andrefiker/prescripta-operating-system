# Prescripta Operating System

Site interno/estrategico da Prescripta para consolidar a arquitetura comercial e operacional da empresa.

Conteudo principal do site:

- tese e posicionamento da empresa
- ICPs priorizados
- oferta, planos e pricing
- CRM, funil comercial, one-pager, deck e script de demo
- mapa juridico e de compliance minimo vendavel
- lista de documentos obrigatorios
- onboarding, implantacao e customer success
- roadmap executivo de 90 dias

## Stack

- React 19
- TypeScript
- Vite 8

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Objetivo do repositorio

Este repositorio nao e o produto da Prescripta. Ele serve como manual operacional em formato web para alinhar socios,
juridico, vendas, implantacao e GTM em torno da mesma estrutura de empresa.
